#  Agentic AI Business Analyst

An autonomous multi-agent system designed to simulate real-world business analysis by automating data processing, insight generation, and reporting.



##  Overview

This project replicates the workflow of a business analyst using multiple intelligent agents. It takes raw data as input and automatically performs data cleaning, machine learning analysis, visualization, and report generation.

The goal is to reduce manual effort and provide quick, data-driven insights.

##  Agents

* Data Cleaning Agent – Handles preprocessing and missing values
* ML Agent – Applies machine learning models
* Insight Agent – Generates business insights
* Visualization Agent – Creates charts
* Report Agent – Generates PDF reports



##  Tech Stack

* Python
* Streamlit
* Scikit-learn
* Plotly
* ReportLab

##  How It Works

1. User uploads dataset
2. Data is processed
3. ML model runs
4. Insights generated
5. Report created
